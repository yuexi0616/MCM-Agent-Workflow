# 碳化硅外延层厚度的双光束和多光束干涉法测量研究

# 摘要

碳化硅是一种新兴的第三代半导体材料，其外延层的厚度对其性能至关重要。而红外干涉法是外延层厚度的一种无损伤测量方法。本文通过分析红外光束的干涉效应，建立了相应的干涉模型和厚度测量模型。采用多种优化方法，在双光束和多光束干涉场景下获得了精确的的外延层厚度值。

针对问题一：构建双光束干涉模型。首先，依据 Snell 定律修正入射角的影响，推导出外延层内光程差与相位差。接着，利用 Fresnel 公式计算空气-外延层、外延层-衬底界面的反射系数，引入 Cauchy 色散模型描述折射率随波长的变化，建立外延层厚度与红外光谱反射率的模型关系。求解时通过傅里叶变换法获取干涉条纹周期并提取振荡周期，得到外延层厚度粗略的初始估计值，最后采用非线性最小二乘法拟合模型参数，为后续厚度反演提供清晰的理论框架。

针对问题二：基于问题一模型，对碳化硅晶圆片在入射角 $10^{\circ}$ 和 $15^{\circ}$ 的实验数据进行厚度反演。首先对光谱数据进行等间距重采样，裁剪 $1000 \mathrm{~cm}^{-1}$ 以下低波数干扰区。然后对光谱利用非对称最小二乘法（AsLS）去趋势与Savitzky Golay平滑去噪。随后通过滑窗傅里叶变换筛选高信噪比有效频段 $[2500, 3700] \mathrm{cm}^{-1}$ ，采用双角联合拟合策略共享厚度与色散参数，引入波数刻度和入射角微偏差校准量，利用随机梯度下降法进行全参拟合，最终得到SiC外延层厚度约 $7.604 \mu \mathrm{m}$ ，残差均方根误差低于 $1.14 \times 10^{-3}$ 。此外，本文从残差统计特性、正态性检验、角度一致性、系统敏感性等多个维度对模型进行了可靠性检验，确保了模型的准确性与稳健性。

针对问题三：构建 Airy 多光束干涉 + Cauchy 色散模型。首先对两角度光谱进行等距重采样与轻量多项式去基线，随后采用滑窗 FFT 提取主频 $f^{*}$ 与信噪比，并据此选择有效带宽 $[1170, 2170]$ cm $^{-1}$ 作为厚度反演的工作区间。结合相干长度判据与界面反射率锐度系数 F 的分析得到多光束相干条件，并据此判定出硅晶圆片存在显著多光束干涉，而碳化硅样品因 SiC/Si 界面反射率仅约 0.016，高阶反射贡献可忽略，沿用两光束模型即可。基于此，我们对硅样品建立 Airy 型多光束干涉 + Cauchy 色散模型，在两角度数据上联合最小二乘全参拟合，残差主峰位置复现实验条纹、幅值整体小于 0.02，最终得到硅晶圆片厚度约 3.814μm。

关键字：外延层厚度测量 傅里叶变换 多光束干涉 Cauchy 色散模型 Airy 公式

# 一、问题重述

# 1.1 问题背景

碳化硅（SiC）作为新一代半导体材料，因其宽禁带、高热导率和高击穿场强等特性，被广泛应用于电力电子、轨道交通、新能源汽车等领域。外延工艺是碳化硅晶圆制造的关键环节，而外延层的厚度直接影响器件的导通电阻、击穿电压和整体性能。因此，如何准确测定外延层厚度，不仅是科研中的重要课题，也是产业界在质量控制与性能优化中的迫切需求。

与此同时，硅材料作为传统半导体的基础，在电子器件制造中依然占据着重要地位。对硅及碳化硅材料外延层的厚度测量与比较研究，不仅能够为半导体产业的发展提供参考数据，也有助于推动材料检测技术的进一步完善。

# 1.2 问题提出

基于上述背景，本文建立数学模型解决下列问题：

问题一 在仅考虑外延层与衬底界面分别发生一次反射与投射所形成的两光束干涉情况下，构建外延层厚度的数学模型。

问题二 利用问题一所建立的模型，对碳化硅晶圆片在入射角为 $10^{\circ}$ 和 $15^{\circ}$ 时的实验数据进行厚度反演，并对结果的准确性与可靠性进行评估。

问题三 分析多光束干涉产生的条件和其对厚度测定结果的影响，判断硅晶圆片的实验数据是否存在多光束干涉，并在必要时建立相应的模型与算法进行厚度求解。同时，若碳化硅数据中存在类似影响，提出相应的修正方法并给出校正后的厚度结果。

# 二、问题分析

# 2.1 问题一

问题一的核心是建立一个光学干涉模型，用于描述外延层厚度与红外反射光谱之间的关系。入射光在外延层表面和衬底界面分别发生反射，两束光在一定条件下相干叠加，从而形成干涉条纹。条纹的周期和强度不仅依赖于外延层的厚度，还与材料折射率和入射角有关。因此，需要在理想化的“两光束干涉”假设下，构建厚度与波数之间的函数关系。由于折射率随波长和载流子浓度而变化，可引入合适的色散模型加以描述，使得模型更贴近实际情况。通过这一模型，可以为后续的厚度估计和算法设计提供清晰的理论依据。

# 2.2 问题二

在已有两光束干涉模型的基础上，问题二的任务是利用实测数据对外延层厚度进行反演计算。已知条件是碳化硅晶圆片在两种入射角下的实测光谱曲线，关键在于如何将模型与实际数据结合，得到可靠的厚度值。

首先，需要对光谱数据进行预处理，包括去除噪声、归一化、选择合适的频段，以确保用于拟合的数据的质量。随后，根据问题一建立的模型，设计合适的算法对参数进行拟合。为了保证结果的稳定性和可信度，可以采用两步法：先通过频域分析获得厚度的初值，再利用非线性优化方法在全频段范围内精细拟合。在此过程中，充分利用入射角 $10^{\circ}$ 与 $15^{\circ}$ 的两组数据，通过联合拟合与结果对比来验证模型的适用性。最后，结合残差分析与参数灵敏度讨论，对所得厚度结果给出可靠性评估。

# 2.3 问题三

问题三在前两问的基础上考虑了多光束干涉的更多情况，要求分析产生多光束干涉的条件以合理选择模型解决相应问题。在建立模型之后，根据硅晶圆片和碳化硅晶圆片的实测光谱曲线，选择合适的模型，得到更精确的效果。

为此，对问题一的模型进行一些修正，将其推广到多光束情形，同时得到必要条件与精度影响。在推导获得多光束模型后，将光谱数据按问题二的流程进行预处理，通过频域分析获取初值，采用非线性优化算法对硅晶圆片用多光束模型进行拟合，从而评价模型的适用性，并给出可靠性评估。对碳化硅晶圆片，在预处理后，利用带通滤波的方法对原图进行优化，消除其他光束的影响，再利用问题二的计算方法得到厚度值。

# 三、模型假设

1. 厚度与面内均匀性：假设外延层厚度 d 在光斑范围内不随波数/时间变化（面内与时间上均匀稳定）。

2. 衬底折射率平稳: 假设在选定带宽内, 衬底折射率 $n_{2}$ 近似平稳。SiC 样本取 $n_{2} \approx 2.65$ , Si 样本取 $n_{2} \approx 3.42$ 。

3. 各向同性：假设介质各向同性、界面理想平行，忽略粗糙、楔角影响。

4. 介质吸收率较低：假设介质足够透明，红外光在介质表面发生折射和反射以及在介质中传播时介质对光能的吸收较低可以忽略，反射率与透射率之和为1。

# 五、模型建立与求解

# 5.1 问题一模型的建立与求解

# 5.1.1 两光束干涉厚度模型的建立

在红外干涉测厚实验中，入射光在空气 外延层界面和外延层 衬底界面各发生一次反射，两束光在探测端叠加产生干涉条纹。为描述这一现象，我们建立“两光束干涉厚度模型”。

设空气折射率为 $n_{0}=1$ ，外延层折射率为 $n_{1}(\lambda)$ ，厚度为 d，衬底折射率为 $n_{2}$ ，入射角为 $\theta_{0}$ 。根据 Snell 定律，外延层中的折射角为

$$
n _ {0} \sin \theta_ {0} = n _ {1} (\lambda) \sin \theta_ {1}. \tag {1}
$$

其中， $\theta_{1}$ 为外延层中的折射角。

光在外延层内的往返光程差为

$$
\Delta L = 2 n _ {1} (\lambda) d \cos \theta_ {1}, \tag {2}
$$

对应的相位差为

$$
\delta (\lambda) = \frac {4 \pi n _ {1} (\lambda) d \cos \theta_ {1}}{\lambda}. \tag {3}
$$

图 1 两光束干涉模型的光路示意图


利用 Fresnel 公式，空气-外延层及外延层-衬底的反射系数分别为：

$$
r _ {0 1} = \frac {n _ {0} \cos \theta_ {0} - n _ {1} (\lambda) \cos \theta_ {1}}{n _ {0} \cos \theta_ {0} + n _ {1} (\lambda) \cos \theta_ {1}}, \tag {4}
$$

$$
r _ {1 2} = \frac {n _ {1} (\lambda) \cos \theta_ {1} - n _ {2} \cos \theta_ {2}}{n _ {1} (\lambda) \cos \theta_ {1} + n _ {2} \cos \theta_ {2}}, \tag {5}
$$

其中 $\theta_{2}$ 为在衬底中的折射角。

则总反射率为

$$
R (\lambda) = \left| r _ {0 1} + r _ {1 2} e ^ {i \delta (\lambda)} \right| ^ {2}. \tag {6}
$$

为刻画色散效应，引入 Cauchy 模型近似外延层折射率：

$$
n _ {1} (\lambda) = A + \frac {B}{\lambda^ {2}}, \tag {7}
$$

其中 A, B 为待定参数。至此，我们已建立了外延层厚度 d 与红外光谱 $R(\lambda)$ 之间的函数关系。

# 5.1.2 两光束干涉厚度模型的求解

由于厚度 d 出现在非线性相位函数中，难以直接解析求解，需要结合实验数据进行数值反演。求解思路如下：

首先由干涉条纹周期与厚度的近似关系

$$
\Delta \left(\frac {1}{\lambda}\right) \approx \frac {1}{2 n _ {1} d \cos \theta_ {1}}, \tag {8}
$$

通过 FFT 分析光谱条纹周期，对厚度做出初步估计，得到厚度初值 $d_{0}$ 。

接下来构造目标函数：

$$
J (d, A, B) = \sum_ {i} \left(R ^ {\mathrm{obs}} (\lambda_ {i}) - R (\lambda_ {i}; d, A, B)\right) ^ {2}, \tag {9}
$$

其中 $R^{obs}$ 为实验数据， $R(\cdot)$ 为理论模型。

下面以 $d_{0}$ 为初值，采用非线性最小二乘法拟合参数 $(d, A, B)$ ，得到厚度估计 $d^{*}$ 。

最后，对不同入射角的两组实验数据进行联合拟合，比较结果的一致性，并通过残差分析验证拟合的合理性。

# 5.2 问题二模型的建立与求解

# 5.2.1 数据预处理

在正式代入数据进行反演之前，要对数据进行预处理。为保证频域分析（FFT）与滑动窗口的有效性，需要在波数 $\sigma$ 上等间距采样。将原始数据 $(\sigma_{i}, R_{i})$ 线性插值到等间距网格，步长取原始间距的中位数：

$$
\Delta \sigma \approx m e d i a n (\sigma_ {i + 1} - \sigma i). \tag {10}
$$

其中，反射率按小数参与计算： $R = R(\%) / 100$ 。

等间距重采样后得到数据如图2所示。其中 raw 代表原始数据，Uniform Sampling 代表处理之后的数据。

由图2可以看出，在原始光谱中，低波数区域（约 $1000 \, cm^{-1}$ 以下），出现了明显的高反射率平台与陡降，这是由碳化硅材料的声子极化子（Reststrahlen）区引起的物理效应：当波数位于横光学声子（TO）与纵光学声子（LO）频率之间时，介电常数为负，表面反射率接近 1；而在 LO 边附近（约 $970 - 1000 \, cm^{-1}$ ）会出现剧烈的色散与吸收，导致光谱中出现“暴涨 暴跌”的特征。这一区域的信号主要受材料本征吸收主导，而不再体现出稳定的干涉条纹。

将该区间数据保留，会对基线估计和平滑滤波造成干扰，同时在滑窗 FFT 分析中导致主频率与信噪比的不稳定，从而降低厚度估计的可靠性。因此，我们在预处理阶段将 $1000 \, cm^{-1}$ 以下的波数段整体舍弃，仅保留 $1000 - 4000 \, cm^{-1}$ 的有效光谱进行后续分析。裁剪后的数据如图3所示。

接下来，我们对数据进行去趋势化、去噪、平滑处理。首先采用非对称最小二乘法(AsLS)拟合基线 $\hat{b}$ ，扣除了基线信号，有效去除了由强吸收与整体趋势引起的低频分量：

$$
\min _ {z} \sum_ {i} w _ {i} (R _ {i} - z _ {i}) ^ {2} + \lambda \sum_ {i} (\Delta^ {2} z _ {i}) ^ {2}, \quad w _ {i} = \left\{ \begin{array}{l l} p, & R _ {i} > z _ {i} \\ 1 - p, & R _ {i} \leq z _ {i} \end{array} \right. \tag {11}
$$

其中 $\Delta^{2}$ 为二阶差分算子， $\lambda > 0$ 控制平滑度， $p \in (0,1)$ 让基线贴近下包络。

# 5.2.2 基于 Cauchy 折射率模型的外延层厚度反演

物理模型与参数化与校准量 设两组入射角分别为 $\theta_{0}^{(10)} = 10^{\circ}$ 与 $\theta_{0}^{(15)} = 15^{\circ}$ ，对应观测数据为 $\{(\bar{\nu}_{k,i}, R_{k,i}^{\mathrm{obs}})\}$ ，其中 $\bar{\nu} = 1/\lambda$ 为波数， $k \in \{10, 15\}$ 标识角度。

为消除仪器与几何的系统性微偏差，引入两类校准量：

1. 波数刻度校准 $\varepsilon$ ：令真实波数为 $\tilde{\nu}^{\mathrm{true}} = (1 + \varepsilon)\tilde{\nu}$ ，以补偿谱仪刻度的线性误差。

2. 入射角微偏差 $\Delta\theta_{k}$ ：令有效入射角 $\tilde{\theta}_{k}=\theta_{0}^{(k)}+\Delta\theta_{k}$ ，以补偿装调误差。

考虑上述校准后，两光束模型的理论反射率写为

$$
R _ {\text { model }} \left(\tilde {\nu}; d, A, B, \tilde {\theta} _ {k}, (1 + \varepsilon)\right) = \mathcal {R} \left(\lambda = \tilde {\nu} ^ {- 1}, d, n _ {1} (\lambda ; A, B), \tilde {\theta} _ {k}\right), \tag {19}
$$

其中 $\mathcal{R}(\cdot)$ 由第5.1.1节给出（Fresnel强度反射率 $R_{1}, R_{2}$ 与相位差 $\delta = \frac{4\pi n_{1}d\cos\theta_{1}}{\lambda}$ 的组合式）。

$$
n (\lambda) = A + \frac {B}{\lambda^ {2}}, \tag {20}
$$

其中 A, B 为待估参数。

由 Snell 定律有 $\sin\theta_{t}=\sin\theta_{i}/n(\lambda)$ ，从而 $\cos\theta_{t}=\sqrt{1-\sin^{2}\theta_{t}}$ 。

设外延层厚度为 $d[\mu m]$ ，薄膜往返相位为

$$
\beta (\lambda , \theta_ {i}) = \frac {2 \pi n (\lambda) d \cos \theta_ {t}}{\lambda}. \tag {21}
$$

为了将求解重点放在相位对齐（决定条纹位置、对 $d$ 最敏感），我们采用相位优先的反射率模型：

$$
\widehat {R} (\nu ; \theta_ {i}) = c _ {\theta} + s _ {\theta} \cos (2 \beta (\lambda , \theta_ {i})), \tag {22}
$$

其中 $c_{\theta}, s_{\theta}$ 为不同入射角的幅度与偏置系数。

闭式幅值/基线标定 为抵消探测链路的幅值与基线差异，对每条曲线保留闭式线性标定：给定一组参数 $(d, A, B, \varepsilon, \Delta \theta_k)$ ，先计算理论序列 $y_{k,i} = R_{\mathrm{model}}(\tilde{\nu}_{k,i}(1 + \varepsilon); d, A, B, \tilde{\theta}_k)$ ，然后用最小二乘闭式解：

$$
a _ {k} = \frac {\operatorname{Cov} (y _ {k} , R _ {k} ^ {\mathrm{obs}})}{\operatorname{Var} (y _ {k})}, \quad b _ {k} = \overline {{R _ {k} ^ {\mathrm{obs}}}} - a _ {k} \overline {{y _ {k}}}, \tag {23}
$$

得到校正后的模型输出 $\widehat{R}_{k,i}=a_{k}y_{k,i}+b_{k}$ 。这种“内环闭式、外环非线性”的结构可减少优化变量，提升稳定性。

双角联合的鲁棒目标函数 直接分别对 $10^{\circ}$ 与 $15^{\circ}$ 两组数据独立拟合厚度 $d_{10}$ 与 $d_{15}$ ，再取其平均作为最终结果的方法主要存在以下两大缺陷：

1. 物理一致性不足：外延层厚度 d 在物理上应为同一常数，而独立拟合时每个角度的色散参数 $(A, B)$ 可能向不同方向漂移，导致 $d_{10}, d_{15}$ 不完全可比。

2. 稳健性不足：当某一角度数据在局部波段信噪比较低或受吸收峰干扰时，独立拟合结果容易受到显著偏移，取平均并不能消除该偏差。

相比之下，双角联合拟合在一个统一的目标函数下同时利用两角度的观测数据，共享厚度 $d$ 与折射率色散参数 $(A, B)$ ，而仅允许角度校准量 $\Delta \theta_{k}$ 独立变化。这种方式既保证了厚度的物理唯一性，又利用了两角度在噪声与系统误差上的互补性，从而得到更稳定、可靠的反演结果。

基于上述考虑，我们构造了带有物理先验正则的加权鲁棒最小二乘目标：

$$
\min _ {d, A, B, \varepsilon , \{\Delta \theta_ {k} \}} \sum_ {k \in \{1 0, 1 5 \}} \sum_ {i} w _ {k, i} \rho \left(R _ {k, i} ^ {\mathrm{obs}} - \widehat {R} _ {k, i}\right) + \underbrace {\eta_ {\varepsilon} \varepsilon^ {2} + \eta_ {\theta} \sum_ {k} \left(\Delta \theta_ {k}\right) ^ {2} + \eta_ {A} \left(A - A _ {0}\right) ^ {2} + \eta_ {B} \left(B - B _ {0}\right) ^ {2}} _ {\text {物理先验正则}}. \tag {24}
$$

其中 $w_{k,i}$ 为数据可信度权重（在吸收段/低信噪比段取较小值）， $\rho(\cdot)$ 为鲁棒损失函数，本文选用 soft_11 形式； $(A_{0}, B_{0})$ 为基于厚度初值 $d_{0}$ 的预拟合结果， $\eta_{\bullet}$ 为正则权重参数。

这样构造的目标函数兼顾了“内环闭式、外环非线性”的结构：每次迭代中先用闭式解对幅值/基线进行标定，再对 $(d,A,B,\varepsilon,\Delta\theta_{k})$ 进行非线性更新，从而在保证收敛稳定性的同时减少了优化变量的数量。

两阶段初始化 在确立了目标函数之后，我们首先采用有效范围内全谱段（约 $2500 \sim 3800 \, cm^{-1}$ ）进行直接拟合。对相关参数做两阶段初始化：

1. 频域初值：由干涉条纹主频得 $d_{0}d_{0,k}=\left(2\hat{f}_{0,k}\sqrt{n^{2}-\sin^{2}\theta_{0}^{(k)}}\right)^{-1}$ ，将两角合并取加权平均得 $d_{0}$ 。

2. $(A_{0}, B_{0})$ 预拟合：固定 $d = d_{0}$ ，并在每条曲线内用公式(23)的闭式 $(a_{k}, b_{k})$ 对齐后，仅对 $(A, B)$ 做轻量最小二乘，得到 $(A_{0}, B_{0})$ ，为后续全参拟合提供合理先验。

约束与求解器 为保证物理合理与数值稳定，设置有界约束

$$
d \in [ 1, 5 0 ] \mu \mathrm{m}, \quad A \in [ 2. 3, 2. 8 ], \quad B \in \left[ B _ {\min}, B _ {\max} \right], \quad | \varepsilon | \leq 2 \times 1 0 ^ {- 3}, \quad | \Delta \theta_ {k} | \leq 0. 3 ^ {\circ},
$$

其中 B 的范围根据材料与谱段经验选取（实现中用 $[-0.05, 0.20]$ 或宽松上界并配合正则拉回）。

以 $(d_0, A_0, B_0, \varepsilon = 0, \Delta \theta_k = 0)$ 为初值，采用带界的Levenberg-Marquardt/信赖域算法求解目标函数(24)，并在启动前对 $d$ 在 $[d_0(1 \pm 5\%)]$ 作一次一维粗搜索，以降低局部极小风险。

拟合 我们的拟合流程如算法1所示。

# 5.3 问题三模型的建立与求解

# 5.3.1 多光束干涉的必要条件

在实际实现中，光波可以在外延层界面和衬底界面产生多次反射和透射，从而产生多光束干涉。

设外延层折射率为 $n(\lambda)$ 、厚度为 d，入射角为 $\theta_{0}$ ，层内折射角为 $\theta$ （由斯涅尔定律给出）。记外表面（空气/外延层）与内表面（外延层/衬底）的复反射系数为 $r_{1}, r_{2}$ ，透射系数为 $t_{1}, t_{2}$ 。各级反射光的复振幅为：

$$
\mathbf {E} _ {1} = \mathbf {E} _ {0} r _ {1}, \mathbf {E} _ {m} = \mathbf {E} _ {0} t _ {1} t _ {1} ^ {\prime} r _ {2} e ^ {i \delta} \left(r _ {1} ^ {\prime} r _ {2} e ^ {- i \delta}\right) ^ {m - 2}, \quad (m = 2, \dots), \tag {25}
$$

其中相位推移

$$
\delta (\lambda) = \frac {4 \pi n (\lambda) d \cos \theta}{\lambda} = 4 \pi n (\sigma) d \cos \theta \sigma , \quad (\sigma = 1 / \lambda). \tag {26}
$$

总场为级数之和：

$$
\mathbf {E} _ {r} = \mathbf {E} _ {0} r _ {1} + \mathbf {E} _ {0} t _ {1} t _ {1} ^ {\prime} r _ {2} e ^ {i \delta} \sum_ {m = 2} ^ {\infty} (r _ {1} ^ {\prime} r _ {2} e ^ {- i \delta}) ^ {m - 2} = \mathbf {E} _ {0} \left(r _ {1} + \frac {(1 - r _ {1} ^ {2}) r _ {2} e ^ {- i \delta}}{1 - r _ {1} r _ {2} e ^ {- i \delta}}\right). \tag {27}
$$

由此得到反射率的 Airy 型表达式：

$$
R (\lambda) = \frac {R _ {1} + R _ {2} + 2 \sqrt {R _ {1} R _ {2}} \cos \delta}{1 + R _ {1} R _ {2} + 2 \sqrt {R _ {1} R _ {2}} \cos \delta}, \quad R _ {i} = | r _ {i} | ^ {2}. \tag {28}
$$

该式表明：条纹的相位由 $\delta$ 决定，线形与峰锐度由 $R_{1}, R_{2}$ 决定。

故产生多光束干涉需要满足以下条件：

1. 相干长度条件：光源相干长度 $L_{c}$ 应大于往返光程差，

$$
L _ {c} \gtrsim 2 n d \cos \theta . \tag {29}
$$

2. 反射率条件：界面反射率不能过小，需保证 $\left|r_{1}r_{2}\right| \leqslant 1$ 。锐度系数

$$
F = \frac {4 \sqrt {R _ {1} R _ {2}}}{(1 - \sqrt {R _ {1} R _ {2}}) ^ {2}} \tag {30}
$$

不应太小。

3. 低吸收条件：外延层吸收系数 $\alpha$ 较小，使得 $e^{-2\alpha d\cos\theta}$ 不过小，否则高阶光被显著削弱。

4. 几何与分辨条件：入射光束平行性好，样片楔角小，仪器分辨率优于条纹间隔。

# 5.3.2 多光束干涉对厚度计算精度可能的影响

设条纹相位为 $\delta = 4\pi nd \cos \theta / \lambda$ 。

1. 在理想对称、低损耗条件下，极值近似满足 $\delta = 2\pi m$ ，条纹间隔

$$
\Delta \sigma \approx \frac {1}{2 n d \cos \theta} \tag {31}
$$

与两光束模型一致。因此，多光束干涉并不显著改变条纹的基本周期。

2. 多光束干涉会使条纹更尖锐、对比度增强。如果仍使用两光束正弦模型拟合，会出现模型失配，导致残差具有结构性，从而引入系统偏差。

3. 当 $R_{1} \neq R_{2}$ 或外延层存在吸收时，极值位置发生微小漂移，若用两光束近似会进一步放大厚度计算误差。

4. 在傅里叶域，多光束干涉会引入谐波分量，若忽略处理，可能造成厚度估计偏差。

综上，多光束干涉的存在会影响厚度计算的精度，需要在建模与拟合过程中予以考虑。

考虑到低波数段噪声与仪器畸变相对明显，我们仅保留 $500 \sim 4000 \, cm^{-1}$ 的范围作为有效工作带。

基线校正 我们采用轻量化的多项式拟合去基线方法：对归一化反射率 $R(\sigma)$ 与波数 $\sigma$ 进行一次或二次多项式拟合

$$
R _ {\mathrm{trend}} (\sigma) \approx a _ {0} + a _ {1} \sigma \quad (\text {或} a _ {0} + a _ {1} \sigma + a _ {2} \sigma^ {2}), \tag {33}
$$

再将其从原始曲线上扣除，得到去趋势后的干涉信号

$$
R _ {\text { detrended }} (\sigma) = R (\sigma) - R _ {\text { trend }} (\sigma). \tag {34}
$$

与复杂的基线校正算法相比，该方法简洁且能够保留干涉条纹的相位信息，避免过度滤波导致条纹畸变。

滑窗 FFT 与信噪比评估 为验证条纹周期的稳定性与预处理效果，采用汉宁窗的滑动窗口快速傅里叶变换（Sliding-FFT）提取局部主频 $f^{*}$ 与信噪比 SNR。窗口宽度 W 取 $1000 \, cm^{-1}$ （覆盖 $\geq 4$ 个条纹周期），步长 $150 \, cm^{-1}$ 。对每个窗信号 x，先行去均值并乘以窗函数 w，计算

$$
Y = \left| \operatorname{FFT} (w \cdot (x - \bar {x})) \right|, \tag {35}
$$

主峰幅度记为 $A_{pk}$ ，噪声幅度取主峰左右排除若干频点后的均方根，得到 SNR = $A_{pk}/RMS(noise)$ 。

# 5.3.4 多光束干涉厚度模型的建立

下面，我们建立多光束干涉下的反射率模型。设所有反射光振幅叠加为：

![image](https://cdn-mineru.openxlab.org.cn/result/2026-05-27/ef3a68d3-b96b-4f83-b657-9986d6c2de3d/932494dd1f4a069384849738cd2c774c695b099878730b0c6d70e327a082db5e.jpg)



图 28 两光束干涉模型的光路示意图


$$
\mathbf {E} _ {r} = \mathbf {E} _ {0} (r _ {1} + t _ {1} t _ {1} ^ {\prime} r _ {2} e ^ {- i \delta} + t _ {1} t _ {1} ^ {\prime} r _ {1} ^ {\prime} r _ {2} ^ {2} e ^ {- 2 i \delta} + \dots + t _ {1} t _ {1} ^ {\prime} r _ {1} ^ {m - 2} r _ {2} ^ {m - 1} e ^ {- (m - 1) i \delta} + \dots), \tag {36}
$$

其中 $\delta=\frac{4\pi nd\cos\theta}{\lambda}$ 为相位差。

由于 $|r_{1}r_{2}|<1$ ，上述级数收敛，可视作几何级数求和，得到

$$
\mathbf {E} _ {r} = \mathbf {E} _ {0} \left(r _ {1} + \frac {(1 - r _ {1} ^ {2}) r _ {2} e ^ {- i \delta}}{1 - r _ {1} r _ {2} e ^ {- i \delta}}\right). \tag {37}
$$

由此可得反射率

$$
R = \frac {\left| \mathbf {E} _ {r} \right| ^ {2}}{\left| \mathbf {E} _ {0} \right| ^ {2}} = \frac {r _ {1} ^ {2} + r _ {2} ^ {2} + 2 r _ {1} r _ {2} \cos \delta}{1 + r _ {1} ^ {2} r _ {2} ^ {2} + 2 r _ {1} r _ {2} \cos \delta} = \frac {(r _ {1} + r _ {2}) ^ {2} - 4 r _ {1} r _ {2} \sin^ {2} \frac {\delta}{2}}{(1 + r _ {1} r _ {2}) ^ {2} - 4 r _ {1} r _ {2} \sin^ {2} \frac {\delta}{2}}. \tag {38}
$$

这就是典型的 Airy 干涉公式，与 Fabry-Pérot 干涉腔形式一致。为便于分析，记 $R_{1}=r_{1}^{2}, R_{2}=r_{2}^{2}$ ，则有

$$
R = \frac {R _ {\max} - F \sin^ {2} \frac {\delta}{2}}{1 - F \sin^ {2} \frac {\delta}{2}}, \tag {39}
$$

其中 F 称为锐度系数，这里定义为

$$
F = \frac {4 \sqrt {R _ {1} R _ {2}}}{(1 + \sqrt {R _ {1} R _ {2}}) ^ {2}}, \tag {40}
$$

并有

$$
R _ {\mathrm{max}} = \left(\frac {\sqrt {R _ {1}} + \sqrt {R _ {2}}}{1 + \sqrt {R _ {1} R _ {2}}}\right) ^ {2}, \quad R _ {\mathrm{min}} = \left(\frac {\sqrt {R _ {1}} - \sqrt {R _ {2}}}{1 - \sqrt {R _ {1} R _ {2}}}\right) ^ {2}. \tag {41}
$$

界面反射率 $R_{1}, R_{2}$ 可由菲涅耳公式计算：

$$
R _ {1} = \frac {1}{2} \left(\frac {\sin^ {2} (\theta_ {0} - \theta_ {1})}{\sin^ {2} (\theta_ {0} + \theta_ {1})} + \frac {\tan^ {2} (\theta_ {0} - \theta_ {1})}{\tan^ {2} (\theta_ {0} + \theta_ {1})}\right), \tag {42}
$$

$$
R _ {2} = \frac {1}{2} \left(\frac {\sin^ {2} (\theta_ {1} - \theta_ {2})}{\sin^ {2} (\theta_ {1} + \theta_ {2})} + \frac {\tan^ {2} (\theta_ {1} - \theta_ {2})}{\tan^ {2} (\theta_ {1} + \theta_ {2})}\right). \tag {43}
$$

由以上推导可见，多光束干涉下的反射率由相位 $\delta$ 控制。条纹极大值与极小值分别对应 $R_{max}$ 和 $R_{min}$ ，条纹锐度取决于精细度系数 F。当 F 较大时，条纹峰值尖锐，厚度分辨率提高，但若仍采用两光束近似拟合，会产生系统偏差；当 F 较小时，多光束效应减弱，可近似为两光束模型。

值得注意的是，厚度 d 与相位 $\delta$ 的关系依然保持周期性：

$$
\delta = \frac {4 \pi n d \cos \theta}{\lambda} \quad \Rightarrow \quad \Delta \sigma \approx \frac {1}{2 n d \cos \theta}. \tag {44}
$$

因此，厚度的主周期估算公式与两光束情况一致，但由于 Airy 线形导致的极值微移和非对称性，若直接使用两光束模型，将低估计算的不确定度，并可能引入 1% ~ 2% 的系统偏差。为获得高精度厚度，应在建模和拟合中采用 Airy 型多光束模型。

# 5.3.5 多光束干涉厚度模型的求解

Step1 数据预处理 数据预处理的工作已经在 5.3.3 完成，此节不再赘述。

Step2 FFT 粗估厚度 在多光束干涉的情形下，干涉条纹依旧表现为近似周期性。我们仍然对预处理后的反射率光谱 $R(\nu)$ （ $\nu$ 为波数，单位 cm $^{-1}$ ）进行快速傅里叶变换（FFT），从频谱中提取主频率 $f_{0}$ 。该频率与光程差的关系为

$$
d _ {0} \approx \frac {f _ {0}}{2 n _ {\mathrm{eff}} \cos \theta}, \tag {45}
$$

其中 $n_{eff}$ 为外延层的有效折射率， $\theta$ 为入射角。

代入数据后得到结果如图29。 $10^{\circ}$ 下估测得到厚度为 $3.675\ \mu m$ ， $15^{\circ}$ 下估测得到厚度为 $3.681\ \mu m$ ，平均为 $3.678\ \mu m$ 。该厚度初值将作为后续基于Cauchy模型的非线性拟合的初始条件，以保证迭代的收敛性与稳定性。

Step3 多光束干涉 + Cauchy 模型拟合 为了同时考虑多光束干涉效应与折射率色散，我们采用基于 Cauchy 模型的 Airy 拟合。折射率表示为

$$
n (\lambda) = A + \frac {B}{\lambda^ {2}}, \tag {46}
$$

其中 A, B 为色散参数。

将其代入多光束干涉反射率公式，可得：

$$
R (\nu ; d, A, B, \varepsilon , \Delta \theta_ {1}, \Delta \theta_ {2}) = f (\nu , d, n (\lambda), \theta + \Delta \theta_ {i}, \varepsilon), \tag {47}
$$

其中 d 为外延层厚度， $\varepsilon$ 表示波数刻度的微小修正， $\Delta\theta_{1},\Delta\theta_{2}$ 分别为两个实验角度的修正量。

在数值实现上，我们采用最小二乘优化，并在每条曲线上引入线性缩放因子 $a_{i}, b_{i}$ ，通过残差函数

$$
\mathrm{res} = \sum_ {i} \left[ a _ {i} R _ {\text { hat }} (\nu ; \cdot) + b _ {i} - R _ {i} ^ {\exp} (\nu) \right] ^ {2} \tag {48}
$$

作为目标函数进行迭代求解。

为提高稳健性，我们优化中引入了对 $(A,B,\varepsilon,\Delta\theta)$ 的正则化约束，避免参数偏离物理合理区间。

在 $10^{\circ}$ 与 $15^{\circ}$ 两个入射角数据的联合拟合下，得到的拟合结果为：

$$
\hat {d} = 3. 8 1 4 \mu \mathrm{m}, \quad \hat {A} = 3. 4 4 5, \quad \hat {B} = 0. 0 0 1.
$$

R _ {1} = \left(\frac {n _ {\mathrm{SiC}} - 1}{n _ {\mathrm{SiC}} + 1}\right) ^ {2} \approx 0. 2 0, \tag {49}

R _ {2} = \left(\frac {n _ {\mathrm{Si}} - n _ {\mathrm{SiC}}}{n _ {\mathrm{Si}} + n _ {\mathrm{SiC}}}\right) ^ {2} \approx 0. 0 1 6. \tag {50}

# 六、模型评价

# 6.1 模型优点

1. 物理完备性：双光束干涉厚度计算模型和多光束干涉厚度计算模型均是基于电磁场理论的严格推导，具有很强的适用性。理论原理与逻辑完备，模型拟合精度较高，获得的结果具有很好的可靠性。

2. 可移植性：更换相应参量后根据折射率大小选取模型，可用于解决各类等倾干涉的问题，具有较大的适用范围。

3. 厚度估计可参考性大: 采用傅里叶分析的方法获取 $\Delta \nu$ , 有效结合了全谱的数据, 结果可靠, 由此推算得到的厚度足够精确, 作为初值进行拟合时能降低成本, 增加计算效率。

# 6.2 模型缺点

1. 理论模型实现难度大: 模型理论严格导致具有较多参量以及繁杂的表达式, 且光谱数据量大, 拟合时的计算复杂度较大, 运行成本较高, 在实用层面可能有所欠缺。而为了降低优化算法的成本，容易导致拟合时有较强的初值敏感性，且容易陷入局部最优，因此需要设定较好的初始值，并做好残差分析和多参数交叉验证。

2. 未考虑外延层对光的吸收：两模型均假设外延层足够透明，对光能的吸收率较小以至于可以忽略。在本征吸收频率附近时两模型已不再适用，需考虑原子光谱等更加复杂的修正。

```python
def make_uniform_sigma(sigma, R, step=None):

    sigma = sigma[idx]
```

```python
def baseline_asls(y, lan=1e5, p=0.001, niter=10):
    L = len(y)
    D = sparse.diags([1, -2, 1], [0, -1, -2], shape=(L - 2, L))
    D = D.T @ D
    w = np.ones(L)
    for _ in range(niter):
    W = sparse.diags(v, 0, shape=(L, L))
    Z = W + lan * D
    z = sparse.linalg.spsolve(Z, v * y)
    w = p * (y > z) + (1 - p) * (y <= z)
    return z 
```

```python
def hampel_filter(x, k=3, t=3.0):
    x = x.copy()
    n = len(x)
    for i in range(n):
    lo, hi = max(0, i - k), min(n, i + k + 1)
    win = x[lo:hi]
    med = np.median(win)
    mad = np.median(np.abs(win - med)) + 1e-12
    if np.abs(x[i] - med) > t * 1.4826 * mad:
    x[i] = med
    return x
```

```python
def parse_tag_angle(name: str):
    n = name.lower()
    tag = "SiC" if "sic" in n else (
    "Si" if re.search(r'(?i)(^|_)si(?!c)', n) else "unknown")

def cauchy_n_lambda(lam_um, A, B):
    return A + B / (lam_um**2 + 1e-18)

def cauchy_piecewise(lambda_um, A1, B1, A2, B2, lambda_star):
    n1 = A1 + B1 / (lambda_um**2 + 1e-18)
    n2 = A2 + B2 / (lambda_um**2 + 1e-18)
    return np.where(lambda_um >= lambda_star, n1, n2)

def soft_C1_penalty(A1, B1, A2, B2, lambda_star, w0=1e3, w1=1e3):
    n1 = A1 + B1 / lambda_star**2
    n2 = A2 + B2 / lambda_star**2

    dn1 = -2 * B1 / (lambda_star**3)
    dn2 = -2 * B2 / (lambda_star**3)
```

```python
def parse_tag_angle(name: str):
    n = name.lower()
    tag = "SiC" if "sic" in n else (
    "Si" if re.search(r'(?i)(~|_)si(?!c)', n) else "unknown")
    m = re.search(r'(?i)(?:sic|si)[--]?(\d{1,2})', n)
    ang = float(m.group(1)) if m else None
    return tag, ang

def cauchy_n_lambda(lam_um, A, B):
    return A + B / (lam_um**2 + 1e-18)

def snell_theta(n_from, n_to, th_from):
    s = (n_from * np.sin(th_from)) / np.maximum(n_to, 1e-18)
    s = np.clip(np.real_if_close(s), -0.999999, 0.999999)
    return np.arcsin(s)

def R_hat(nu_cm1, theta0_deg, d_um, A, B, n2_cst=2.65):
    nu = np.asarray(nu_cm1, float)
    lam = 1e4 / nu #
    n1 = cauchy_n_lambda(lam, A, B)
    n2 = n2_cst

    th0 = np.deg2rad(theta0_deg)
    th1 = snell_theta(1.0, n1, th0)
    th2 = snell_theta(n1, n2, th1)

    Rs01 = (np.sin(th0 - th1) / np.sin(th0 + th1)**2)
    Rp01 = (np.tan(th0 - th1) / np.tan(th0 + th1)**2)
    R1 = 0.5 * (Rs01 + Rp01)

    Rs12 = (np.sin(th1 - th2) / np.sin(th1 + th2)**2)
    Rp12 = (np.tan(th1 - th2) / np.tan(th1 + th2)**2)
    R2 = 0.5 * (Rs12 + Rp12)

    F = 4 * np.sqrt(R1 * R2) / (1 + np.sqrt(R1 * R2)**2)
    Rmax = ((np.sqrt(R1) + np.sqrt(R2)) / (1 + np.sqrt(R1 * R2)))**2
    Rmin = ((np.sqrt(R1) - np.sqrt(R2)) / (1 - np.sqrt(R1 * R2)))**2

    delta = (2 * np.pi / lam) * (2 * n1 * d_um * np.cos(th1))
    half_delta = delta / 2
    R = (Rmin + F * np.sin(half_delta)**2) / (1 - F * np.sin(half_delta)**2)
    return np.clip(R, 0.0, 1.2)
```
