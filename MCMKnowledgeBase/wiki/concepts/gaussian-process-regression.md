# 高斯过程回归与遗传算法两阶段框架

## 问题场景

C 题问题二：已知 BMI 是影响达标时间的主要因素，需要为不同 BMI 分组的孕妇推荐最佳 NIPT 检测时点。核心挑战：(1) 原始观测稀疏且含噪，需先补全每位孕妇的达标时间；(2) 需要在连续 BMI 上寻找最优分割点和最佳时点。

## 两阶段框架

### 阶段一：高斯过程回归（GPR）— 补全个体达标时间

GPR 是一种非参数的贝叶斯方法，不仅给出预测值，还给出预测的**不确定性（后验方差）**。论文用 GPR 为每位孕妇推算最早达标孕周 T_attain 的后验分布。相比确定性回归，GPR 的后验方差可用于评估预测的可靠性。

### 阶段二：遗传算法（GA）— 求解分组与最佳时点

将问题形式化为组合优化：
- 决策变量：k 个 BMI 分组边界 + 每组的最佳检测时点 T_j^opt
- 目标函数：最小化潜在风险（失败率 + 过晚检测风险）
- 约束：检测时点 ≥ 各组 T_attain 的某分位数

GA 适合此类离散-连续混合优化。论文还分析了不同准确率要求下分组和时点的变化——准确率 50% → 11-12 周，90% 以上 → 16-20 周。

## 通用价值

"GPR 生成后验 → 输入 GA 优化决策"是用 ML 代理模型驱动决策优化的通用范式。特别适合目标函数难以显式表达、需要通过数据模型模拟的场景。
