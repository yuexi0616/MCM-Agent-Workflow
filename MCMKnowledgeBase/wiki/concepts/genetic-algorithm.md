# 遗传算法与进化优化方法

## 2025 年赛题中的应用

遗传算法（GA）和差分进化（DE）是本届 MCM 中最频繁出现的优化方法：

| 题目 | 方法 | 用途 |
|------|------|------|
| A 题问题三 | 差分进化 | 三枚烟幕弹投放/起爆时间优化 |
| A 题问题二 | 粒子群算法（PSO） | 验证多起点变步长搜索结果 |
| C 题问题二 | 遗传算法（GA） | BMI 分组边界 + 最佳检测时点联合优化 |
| C 题问题三 | 遗传算法（GA） | 多维特征下的分组与检测时点优化 |

## GA 在 C 题的典型用法

目标：寻找最优的 BMI 分组边界和每组的最佳 NIPT 时点。

- **编码**：染色体 = [边界1, 边界2, ..., 时点1, 时点2, ...]，实数编码
- **适应度**：总潜在风险（失败率 + 过晚检测风险），取负值最大化
- **约束处理**：边界单调递增、时点在合理孕周范围内
- **输出**：不同准确率要求下的分组方案

## 方法选择建议

- **连续变量优化、中等维度** → 差分进化（收敛更快）
- **混合离散-连续、有约束** → 遗传算法（编码灵活）
- **有良好初值、需要精细搜索** → 多起点变步长搜索
- **需要交叉验证** → 至少用两种方法互验（如论文中的 DE ↔ PSO）

论文中使用"主方法求解 + 备选方法验证"的策略值得学习——既不影响求解，又增强了可信度。
