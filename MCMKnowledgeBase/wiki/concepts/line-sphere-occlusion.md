# 线段-球体相交遮挡判定

## 核心思想

在军事烟幕遮蔽等场景中，判断一个目标是否被遮蔽可转化为几何问题：若从点光源（导弹）到目标表面任意一点的线段都与遮蔽球体（烟幕云团）相交，则目标被完全遮蔽。

## 数学模型

将导弹位置 P₁ 视为点光源，目标表面点 P₂，云团球心 C、半径 r。线段参数方程代入球面方程：

$$s^2(\mathbf{d}\cdot\mathbf{d}) + 2s(\mathbf{d}\cdot\mathbf{OC}) + (\mathbf{OC}\cdot\mathbf{OC}) - r^2 = 0$$

其中 d = P₂ - P₁，OC = P₁ - C。

## 判别条件

根据一元二次方程判别式 Δ = b² - 4ac，结合根的范围判断：

| 判别式 | 几何含义 |
|--------|----------|
| Δ < 0 | 直线与球无交点 |
| Δ = 0 | 直线与球相切 |
| Δ > 0 | 直线贯穿球体 |

还需验证：至少一个根 s ∈ [0,1]（交点在线段上）或 s₁ < 0 且 s₂ > 1（线段完全在球内）。当目标圆柱上**所有点** P₂ 都满足上述条件时，方为有效遮蔽。

## 效率优化

论文证明了两条关键引理以加速计算：
1. 有效遮蔽时间区间是**连续区间**（因运动速度恒定，遮蔽状态不会突变）
2. 只需判断圆柱体**上下底面圆周**是否被完全遮蔽，即可判定整个圆柱体

离散化验证：圆周上取 300 个离散点即可兼顾精度和效率。
