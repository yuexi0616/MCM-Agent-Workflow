# 双光束干涉与厚度测量模型

## 物理原理

红外光束在空气-外延层界面和外延层-衬底界面各发生一次反射。两束反射光的光程差产生相位差，叠加后形成干涉条纹。条纹周期由外延层厚度 d 和折射率 n₁ 决定。

## 模型构建步骤

1. **光程差**：ΔL = 2n₁d·cos(θ₁)，其中 θ₁ 由 Snell 定律 n₀sin(θ₀) = n₁sin(θ₁) 确定
2. **相位差**：δ = 4πn₁d·cos(θ₁)/λ
3. **反射系数**：Fresnel 公式计算 r₀₁（空气-外延层）和 r₁₂（外延层-衬底）
4. **总反射率**：R(λ) = |r₀₁ + r₁₂·e^(iδ)|²
5. **色散模型**：Cauchy 公式 n₁(λ) = A + B/λ² 引入待定参数 A, B

## 厚度反演的两步法

**粗估计（频域）**：干涉条纹周期与厚度近似关系 Δ(1/λ) ≈ 1/(2n₁d·cosθ₁)，通过 FFT 提取主频得到 d₀。

**精细拟合（时域）**：以 d₀ 为初值，构造目标函数 J(d,A,B) = Σ(R_obs - R_model)²，使用非线性最小二乘法（或随机梯度下降）全参数拟合。

## 多光束扩展

当衬底反射率较高时，光束会在外延层内多次反射，需用 Airy 公式替代两光束模型：

$$R = \frac{r_{01}^2 + r_{12}^2 + 2r_{01}r_{12}\cos\delta}{1 + r_{01}^2r_{12}^2 + 2r_{01}r_{12}\cos\delta}$$

判断依据：相干长度 L_c 与 2n₁d 的关系，以及界面反射率锐度系数 F。SiC/Si 界面反射率仅约 0.016，高阶反射可忽略；Si 样品则需要多光束模型。
