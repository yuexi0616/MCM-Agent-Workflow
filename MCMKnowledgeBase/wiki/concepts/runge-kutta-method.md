# 四阶龙格库塔法 (Runge-Kutta)

龙格库塔法是求解常微分方程初值问题的经典数值方法。四阶龙格库塔法（RK4）截断误差为 $O(h^5)$，是精度与效率的平衡点。

## 递推公式

对于 $\dot{y} = f(t, y), y(t_0) = y_0$：

$$k_1 = f(y_n, t_n)$$
$$k_2 = f(y_n + \frac{h}{2}k_1, t_n + \frac{h}{2})$$
$$k_3 = f(y_n + \frac{h}{2}k_2, t_n + \frac{h}{2})$$
$$k_4 = f(y_n + h k_3, t_n + h)$$
$$y_{n+1} = y_n + \frac{h}{6}(k_1 + 2k_2 + 2k_3 + k_4)$$

时间中点的斜率（$k_2$, $k_3$）权重最大。MATLAB 中 `ode45` 即基于此方法。

## MCM 应用

- **2022 A 题**：波浪能装置垂荡+纵摇的二元二阶→四元一阶微分方程组，用 RK4 迭代求解 40 个波浪周期内 0.2 s 间隔的位移和速度（步长 $h=0.0001$）
- 需将高阶方程降阶为一阶方程组：引入 $u=x_f', w=x_z'$ 等中间变量
- 初始条件通常为零位移、零速度（静水平衡）

## 与遗传算法的配合

功率积分函数的离散化 + RK4 求解状态 → 遗传算法搜索最优阻尼参数。功率取稳态段（100~180 s）的梯形积分均值。
