# 2025 MCM A题：无人机烟幕遮蔽策略优化

## 问题背景

烟幕干扰弹通过化学燃烧或爆炸产生气溶胶云团，能够有效吸收、散射或反射红外线、雷达波等电磁波，从而遮蔽后方目标，干扰敌方侦察和制导系统。由于成本低、效益高，烟幕弹在军事上得到了广泛应用。本题研究无人机投放烟幕弹干扰导弹定位的最优策略。

核心场景：3 枚导弹以 300m/s 匀速直线飞向假目标（原点），5 架无人机在原点前方呈扇形拦截带分布，需要投放烟幕弹形成有效遮蔽。假目标为底面半径 7m、高 10m 的圆柱体。

## 五问递进结构

1. **单机单弹给定参数**：在给定初始参数下，计算烟幕弹对导弹 M1 的有效遮蔽时长
2. **单机单弹策略优化**：以无人机航向、速度、投放时间、起爆时间为决策变量，最大化有效遮蔽时间
3. **单机多弹时序优化**：同一无人机投放多枚烟幕弹，协调时序延长总遮蔽时间
4. **多机单弹空间协同**：三架无人机各投一枚，需避免云团空间过度重叠，保持时间连续
5. **多机多弹多目标**：最复杂场景，需要任务分配（哪架干扰哪枚）+ 资源调度（投几枚、何时投）

## 论文核心方法

- **运动学建模**：为导弹（匀速直线）、无人机（等高匀速）、烟幕弹（抛体运动）、云团中心（匀速下沉 3m/s）分别建立轨迹函数
- **遮挡判定**：将导弹视为点光源，通过线段-球体相交的一元二次方程判别式判断圆柱目标是否被完全遮蔽
- **连续区间证明**：证明有效遮蔽时间区间是连续区间，只需判断圆柱上下底面圆周即可
- **优化算法**：多起点变步长搜索 → 粒子群算法验证 → 差分进化算法 → 分层次优化（先分配后优化，将高维问题拆解）

## 关键结果

| 问题 | 最大有效遮蔽时长 | 方法 |
|------|------------------|------|
| 问题一 | 1.39s | 二分查找 + 300个离散点 |
| 问题二 | 4.59s | 多起点变步长搜索 + PSO验证 |
| 问题三 | 6.94s | 差分进化算法 |
| 问题四 | 11.13s | 预测拦截点 + 12维降维 |
| 问题五 | 21.08s | 分层次优化（任务分配+三个子问题合并） |
