# 2025 MCM B题：碳化硅外延层厚度红外干涉测量

## 问题背景

碳化硅（SiC）是第三代半导体材料，其外延层厚度直接影响器件性能。红外干涉法是一种无损伤测量方法——入射光在空气-外延层界面和外延层-衬底界面分别反射，两束反射光干涉产生条纹，条纹周期包含厚度信息。

数据为碳化硅和硅晶圆片在不同入射角（10° 和 15°）下的红外反射光谱，波数范围约 400-4000 cm⁻¹。

## 三问递进结构

1. **双光束干涉模型**：仅考虑一次反射/透射，构建外延层厚度 d 与光谱反射率 R(λ) 的数学模型
2. **实验数据反演**：对碳化硅实测光谱进行预处理和拟合，反演厚度并对结果做可靠性评估
3. **多光束干涉扩展**：判断硅晶圆片是否存在多光束干涉，建立 Airy 模型进行对比分析

## 论文核心方法

### 模型建立
- **Snell 定律**修正入射角影响 → 推导光程差和相位差
- **Fresnel 公式**计算两个界面的反射系数 r₀₁ 和 r₁₂
- **Cauchy 色散模型** n(λ) = A + B/λ² 描述折射率随波长变化
- 总反射率 R(λ) = |r₀₁ + r₁₂ · e^(iδ)|²

### 数据预处理（严格管线）
1. 波数域等间距重采样 → 裁剪 <1000 cm⁻¹ 的 Reststrahlen 强吸收区
2. **非对称最小二乘法（AsLS）**去基线趋势
3. Hampel 滤波剔除尖噪 + Savitzky-Golay 平滑
4. 滑窗 FFT 分析各频段主频率和 SNR
5. 自动 band 选择：SNR ≥ 10、两角度主频偏差 ≤ 10%

### 厚度反演
- 粗估计：FFT 提取条纹周期 → d₀ 初值
- 精细拟合：双角联合拟合 + 波数刻度/入射角偏差校准 → 随机梯度下降全参优化
- 最终结果：SiC 外延层约 **7.604μm**，残差 RMSE < 1.14×10⁻³

### 模型对比
- SiC/Si 界面反射率仅约 0.016 → 高阶反射可忽略 → 两光束模型足够
- 硅样品 F 系数大 → 需要 Airy 多光束模型 → 硅厚度约 **3.814μm**

## 写作亮点

- 数据预处理步骤有清晰的物理动机（Reststrahlen 区解释）
- "粗估计 + 精拟合"两步法，结合频域和时域
- 多维度可靠性检验：残差统计、正态性检验、角度一致性、系统敏感性
- 对两光束/多光束的选择有物理判据（相干长度 + 锐度系数 F）
