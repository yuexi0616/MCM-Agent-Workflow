# 2025 MCM C题：NIPT检测决策优化与异常识别

## 问题背景

无创产前检测（NIPT）通过分析母体血液中胎儿游离 DNA 筛查染色体异常。核心矛盾：过早检测 → 胎儿 DNA 浓度不足导致失败；过晚检测 → 错过早期干预窗口。cffDNA 浓度受孕周、BMI 等因素显著影响。

数据集包含 1687 条临床记录（男胎 1082 + 女胎 605），31 个特征维度，涵盖 Y/X 染色体浓度、各染色体 Z 值、GC 含量、孕妇 BMI/年龄/身高/体重等。

## 四问递进结构

1. **相关性建模**：分析 Y 染色体浓度与孕周、BMI 的关系，建立关系模型并检验显著性
2. **BMI 分组优化**：基于 BMI 对男胎孕妇分组，给出每组最佳 NIPT 时点
3. **多维特征扩展**：综合身高、体重、年龄等多因素，动态调整最佳时点
4. **女胎异常判定**：无 Y 染色体参考下，基于 21/18/13 号染色体指标判定异常

## 论文核心方法

### 问题一：线性混合效应模型（LMM）
- 用 REML（限制性最大似然估计）处理重复测量的层次结构
- 固定效应：孕周、BMI、身高；随机效应：孕妇个体截距
- 组内相关系数 ICC = 74.3%（个体差异主导总变异）
- 相比传统回归，AIC 提升 17.2%

### 问题二：GPR + 遗传算法两阶段框架
- **阶段一**：高斯过程回归（GPR）为每位孕妇推算最早达标孕周的后验分布
- **阶段二**：遗传算法（GA）求解 BMI 分组边界与最优检测时点的组合优化
- 核心发现：三分组最优，BMI 分界点随准确率要求动态调整
- 50% 准确率 → 11-12 周；90% 以上 → 16-20 周甚至更晚

### 问题三：混合深度生存网络
- **DeepHit** 框架：MLP 处理静态特征（BMI/年龄）+ LSTM 处理时序特征 → 生存分析头输出累积达标概率曲线
- 预测精度 94.7%
- 再次用 GA 优化分组时点，多因素分析发现年龄和 BMI 是最主要静态指标

### 问题四：LightGBM 堆叠集成
- 三个专项模型（13/18/21 号染色体异常各一个）+ 交互影响模型
- Optuna 超参调优 → 元特征 + 原始特征堆叠 → 元学习器
- 二分类准确率 99.7%，3 组独立验证误差 < 1.5%

## 写作亮点

- 每个问题选用的模型有明确的统计/临床动机
- "先理解数据 → 选方法 → 验证假设 → 给结论"的完整叙事
- 随机效应必要性、残差正态性、同方差性等全面诊断
- 模型随问题递进自然升级（LMM → GPR+GA → DeepHit → Stacking）
