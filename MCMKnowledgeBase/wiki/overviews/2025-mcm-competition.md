# 2025 年 MCM 竞赛题目概览

2025 年度共收录 A-E 五道题，涵盖军事优化、光学测量、生物统计、图论仿真、运动科学五个领域。

## 题目对比

| 维度 | A 题：烟幕遮蔽 | B 题：SiC 厚度测量 | C 题：NIPT 优化 | D 题：矿井逃生 | E 题：跳远分析 |
|------|---------------|-------------------|-----------------|---------------|---------------|
| **核心领域** | 军事运筹 | 光学/半导体 | 生物统计/ML | 图论/仿真 | 生物力学 |
| **数学方法** | 几何建模+优化 | 光学模型+参数反演 | 统计建模+ML+优化 | 图网络+最短路径 | 运动学建模 |
| **关键算法** | 差分进化、PSO、变步长搜索 | FFT、非线性最小二乘 | LMM、GPR、GA、DeepHit | BFS、Dijkstra | 物理模型拟合 |
| **编程语言** | MATLAB | Python | Python | Python | Python |
| **优秀论文** | ✓ | ✓ | ✓ | ✓ | 待收录 |
| **数据形式** | 初始参数表 | 光谱曲线 | 临床记录表格 | 巷道网络坐标 | 位置信息+视频 |

## 从建模方法论角度的共同模式

1. **递进式问题链**：每道题都是从简单情况逐步扩展（单→多、静态→动态、低维→高维）
2. **"物理模型 + 数据驱动"融合**：A 题运动学 + 优化、B 题光学 + 统计拟合、C 题医学理论 + ML
3. **多算法交叉验证**：论文中普遍使用 2 种以上的方法互验
4. **可视化贯穿全文**：网络图、光谱对比图、统计分析图

## 优秀论文质量观察

四篇优秀论文的共同特征：
- 符号表统一管理（15-30 个符号，含单位）
- 模型假设逐条列出并说明合理性
- 敏感度分析/鲁棒性检验不是可选项，而是评分要点
- 结果输出到附件（Excel 表格），确保可复现
- 关键词精准概括核心方法（6-8 个）
