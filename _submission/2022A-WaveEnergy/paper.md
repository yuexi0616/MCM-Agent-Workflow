# Summary Sheet

**题目**: 振荡浮子式波浪能发电装置最大输出功率设计

**关键词**: 波浪能; 垂荡-纵摇耦合动力学; 四阶龙格库塔法; 遗传算法; 阻尼优化

---

本研究针对振荡浮子式波浪能发电装置（浮子-振子-PTO系统），建立垂荡与纵摇耦合动力学模型，以最大平均输出功率为目标优化直线与旋转阻尼系数。核心方法为：将四元/八元二阶运动微分方程降阶为一阶系统，采用四阶龙格库塔法（RK4）数值求解，在稳态窗口（100-180 s）内以梯形法计算平均功率，以实数编码遗传算法（GA）搜索最优阻尼参数。GA采用快速模式（种群15、代数30），结果经小步长（h=0.0005 s）交叉验证。

**问题1**（仅垂荡，$\omega=1.4005$ rad/s, $C_h=10000$ N.s/m）: 常数阻尼（$\alpha=0$）下浮子位移幅值约0.19 m，幂次阻尼（$\alpha=0.5$）下约0.21 m；能量平衡闭合相对误差低于0.01%。

**问题2**（垂荡最优阻尼，$\omega=2.2143$ rad/s）: 常数阻尼情形最优直线阻尼系数 $C_h^* = 36620$ N.s/m，最大平均功率 $\overline{P}_{\max} = 193$ W；幂次阻尼情形（$\alpha=0.5$）最优 $C_h^* \approx 78000$ N.s/m。

**问题3**（垂荡+纵摇耦合，$\omega=1.7152$ rad/s, $C_h=10000$ N.s/m, $C_p=1000$ N.m.s）: 稳态平均功率56 W；纵摇角幅值小于0.05 rad，验证小角度假设。

**问题4**（双阻尼协同优化，$\omega=1.9806$ rad/s）: 最优直线阻尼 $C_h^* = 52569$ N.s/m，最优旋转阻尼 $C_p^* = 473585$ N.m.s，最大总平均功率 $\overline{P}_{\text{total}} = 167$ W。旋转阻尼对总功率贡献量级较小——纵摇角速度幅值约 $10^{-3}$ rad/s，导致 $P_p \ll P_h$。

# 1. Introduction

## 1.1 Problem Restatement

振荡浮子式波浪能发电装置（Oscillating Float Wave Energy Converter）由浮子（float）、振子（oscillator）、中轴和PTO系统（能量输出系统，含直线弹簧、直线阻尼器、扭转弹簧和旋转阻尼器）组成。波浪驱动浮子运动，浮子通过PTO带动振子相对运动，阻尼器将相对运动的机械能转化为电能。

本问题要求建立浮子-振子系统的动力学模型，分四个递进阶段进行分析：

- **问题1**: 仅考虑垂荡（heave）自由度，在给定阻尼参数下计算浮子和振子的位移和速度时间序列。分常数阻尼（$\alpha=0$）和幂次阻尼（$\alpha=0.5$）两种情形。
- **问题2**: 以稳态平均输出功率最大化为目标，搜索最优直线阻尼系数 $C_h$。含三个子情形：常数阻尼优化、固定幂指数（$\alpha=0.5$）优化、自由幂指数联合优化。
- **问题3**: 引入纵摇（pitch）自由度，建立垂荡-纵摇耦合的四元二阶微分方程组并求解。
- **问题4**: 同时优化直线阻尼系数 $C_h$ 和旋转阻尼系数 $C_p$，最大化总平均输出功率。

四个问题的波浪圆频率分别为 $\omega = \{1.4005, 2.2143, 1.7152, 1.9806\}$ rad/s，波浪激励力和激励力矩由附件3离散数据经三次样条插值得到。

## 1.2 Literature Review

振荡浮子式波浪能装置（WEC）的动力学建模已有成熟的理论框架。线性波理论（Airy wave theory）下，浮体在垂荡和纵摇方向的运动可表示为含附加质量和辐射阻尼的线性微分方程 [1]。Falnes [2] 系统阐述了WEC的频域与频域分析方法，指出PTO阻尼系数存在使输出功率最大的最优值——这一结论源于线性系统在简谐激励下的阻抗匹配原理。

2022年全国大学生数学建模竞赛A题优秀论文（编号A001）采用大地/固体双坐标系法建立垂荡-纵摇四元二阶耦合方程组，以四阶龙格库塔法（步长0.0001 s）求解，在100-180 s稳态窗口以梯形法积分平均功率，以实数编码GA（种群100、代数200）搜索阻尼参数，得到常数阻尼最优功率229.68 W（$C_h=37639$ N.s/m）、幂次阻尼最优功率230.02 W（$C_h=81478$, $\alpha=0.3377$）[3]。

## 1.3 Our Work

本文在A001方法框架基础上引入以下扩展：

1. **PTO力符号的物理一致性修正**：将PTO弹簧力和阻尼力对浮子（$+K_h x_r + C_h v_{rel}$）和振子（$-K_h x_r - C_h v_{rel}$）的符号统一为"浮子+、振子-"格式，修正了部分文献中符号不一致的问题，并通过能量平衡验证（$\langle dE/dt \rangle \approx 0$）确认其正确性。

2. **振子行程约束**：引入机械限位接触力模型和GA罚函数，防止振子相对位移超出物理允许范围（$\pm 0.8$ m）。

3. **垂荡-纵摇能量平衡完备验证**：推导了包含垂荡和纵摇自由度的完整能量守恒表达式 $dE/dt = u f_e + j M_e - C_{xh} u^2 - C_{xp} j^2 - C_h v_{rel}^2 - C_p \omega_{rel}^2$，并数值验证所有耗散项均为负定，稳态时 $\langle dE/dt \rangle \approx 0$。

4. **时变转动惯量处理**：振子绕纵摇轴的转动惯量 $I_z(t) = I_{z0} + m_z (d_0 + x_r)^2$ 随相对位移 $x_r$ 变化，模型中精确计入。

# 2. Assumptions

1. **海水为理想流体**（无粘、无旋、不可压缩），浮子只受线性微幅波作用。*合理性*：题目明确约定海水为无粘无旋流体，波浪为线性周期微幅波，该假设是线性化建模的工程标准。

2. **忽略中轴、底座、隔层及PTO的质量和摩擦**。*合理性*：题目明确要求忽略；这些组件的质量远小于浮子（4866 kg）和振子（2433 kg），对整体动力学的影响可忽略。

3. **暂态-稳态可分离**：系统在波浪激励下存在暂态响应，约前60 s进入稳态。*合理性*：基于受迫振动理论，线性系统暂态以 $e^{-\zeta \omega_n t}$ 衰减，3-5个时间常数后进入稳态。仿真时长取200 s，稳态窗口取100-180 s。

4. **附加质量和兴波阻尼系数的频率无关近似**：$m' = 1335.535$ kg, $C_{xh} = 656.3616$ N.s/m在四个工作频率下取常数。*合理性*：附件3仅提供一组数值，且工作频率范围较窄（$\omega \in [1.40, 2.21]$ rad/s），该近似引入的系统误差在一阶精度内可接受。

5. **问题1-2中垂荡与纵摇解耦**。*合理性*：垂荡的静水恢复力仅依赖吃水深度变化，与浮子倾角无关；波浪激励力在垂荡方向的分量不受小角度纵摇影响。

6. **问题3-4中小角度纵摇假设**（$|\theta| \leq 5^\circ$），纵摇对垂荡的反向耦合可忽略。*合理性*：$\theta_{\max}=5^\circ$ 时，水线面积相对变化率约0.38%，垂荡恢复力系数B的时变偏差在0.4%以内。

7. **振子为刚体且沿中轴无摩擦滑动**。*合理性*：直线轴承可将摩擦系数降至 $\mu < 0.01$，工程上可忽略。

8. **线性PTO系统假设**：弹簧力与相对位移成正比，阻尼力与相对速度成正比。*合理性*：机械弹簧在小变形范围内线性度良好，液压阻尼器在低速下可近似为线性。

9. **初始时刻系统静置于平衡位置**：$t=0$ 时所有位移和速度均为零。

10. **振子有限行程约束**：相对位移 $x_r \in [-0.8, 0.8]$ m，越界时施加接触刚度罚函数力（$k_{\text{contact}} = 10^6$ N/m²）。*合理性*：实际装置中振子行程受浮子内部结构尺寸限制。

# 4. Model

## 4.1 物理系统描述

振荡浮子式波浪能装置由浮子、振子、中轴和PTO系统构成。PTO系统由直线弹簧（$K_h = 80000$ N/m）、直线阻尼器（系数 $C_h$）、扭转弹簧（$K_p = 250000$ N.m/rad）和旋转阻尼器（系数 $C_p$）组成。浮子受波浪激励力（矩）、附加惯性力（矩）、兴波阻尼力（矩）、静水恢复力（矩）和PTO作用力（矩）。振子仅受PTO作用力和接触力。

## 4.2 垂荡运动方程（问题1）

建立大地坐标系 $OXYZ$（惯性系，$Z$轴向上为正）。浮子和振子的垂荡位移 $x_f(t)$, $x_z(t)$ 均在大地坐标系下度量。

PTO系统对浮子和振子的作用力定义为：

$$
x_r(t) = x_z(t) - x_f(t), \qquad v_{rel}(t) = \dot{x}_z(t) - \dot{x}_f(t)
$$

$$
F_{\text{PTO}\to f} = +K_h x_r + C_h v_{rel}, \qquad
F_{\text{PTO}\to z} = -K_h x_r - C_h v_{rel}
$$

当 $x_r > 0$（振子在浮子上方）时弹簧被拉伸，对浮子产生向上的拉力（正），对振子产生向下的拉力（负）。当 $v_{rel} > 0$（振子比浮子上升更快）时阻尼器阻碍相对运动，对浮子产生向上的推力（正），对振子产生向下的推力（负）。全量符号审计确认所有力项满足牛顿第三定律（见附录A）。

**运动微分方程**：

$$
\begin{cases}
(m_f + m')\,\ddot{x}_f = f\cos\omega t - C_{xh}\dot{x}_f - B x_f + K_h x_r + C_h\,|\dot{x}_r|^{\alpha}\,\text{sgn}(\dot{x}_r) \\[8pt]
m_z\,\ddot{x}_z = -K_h x_r - C_h\,|\dot{x}_r|^{\alpha}\,\text{sgn}(\dot{x}_r)
\end{cases}
$$

其中 $\alpha = 0$ 对应常数阻尼（线性形式 $f_{damp} = C_h v_{rel}$），$\alpha = 0.5$ 对应幂次阻尼（$f_{damp} = C_h |v_{rel}|^{0.5} \text{sgn}(v_{rel})$）。初始条件：$x_f(0) = x_z(0) = \dot{x}_f(0) = \dot{x}_z(0) = 0$.

**降阶为一阶系统**：引入 $u = \dot{x}_f$, $w = \dot{x}_z$，得四元一阶方程组：

$$
\begin{cases}
\dot{x}_f = u \\[4pt]
\dot{x}_z = w \\[10pt]
\dot{u} = \dfrac{1}{m_f + m'}\Big[ f\cos\omega t - C_{xh}u - B x_f + K_h(x_z - x_f) + C_h\,|w-u|^{\alpha}\,\text{sgn}(w-u) \Big] \\[10pt]
\dot{w} = \dfrac{1}{m_z}\Big[ -K_h(x_z - x_f) - C_h\,|w-u|^{\alpha}\,\text{sgn}(w-u) \Big]
\end{cases}
$$

## 4.3 垂荡-纵摇耦合方程（问题3）

纵摇运动引入浮子和振子的角自由度 $\theta_f$, $\theta_z$。振子绕浮子纵摇转轴的转动惯量随相对位移 $x_r$ 变化：

$$
I_z(t) = I_{z0} + m_z\,[d_0 + x_r(t)]^2
$$

PTO扭转弹簧和旋转阻尼器施加的力矩为：

$$
M_{\text{PTO}\to f} = +K_p \theta_r + C_p \omega_{rel}, \qquad
M_{\text{PTO}\to z} = -K_p \theta_r - C_p \omega_{rel}
$$

完整四元二阶方程组（垂荡+纵摇耦合）：

$$
\begin{cases}
(m_f + m')\,\ddot{x}_f = f\cos\omega t - C_{xh}\dot{x}_f - B x_f + K_h x_r + C_h v_{rel} \\[6pt]
m_z\,\ddot{x}_z = -K_h x_r - C_h v_{rel} \\[6pt]
(I_f + I')\,\ddot{\theta}_f = L\cos\omega t - C_{xp}\dot{\theta}_f - L_r\theta_f + K_p\theta_r + C_p\omega_{rel} \\[6pt]
I_z(t)\,\ddot{\theta}_z = -K_p\theta_r - C_p\omega_{rel}
\end{cases}
$$

降阶为八元一阶方程组（引入 $u=\dot{x}_f$, $w=\dot{x}_z$, $j=\dot{\theta}_f$, $k=\dot{\theta}_z$）后，采用四阶龙格库塔法以步长 $h$ 数值求解。

## 4.4 能量平衡验证

机械系统总能量变化率为：

**垂荡情形**：
$$
\frac{dE}{dt} = u\,f_e - C_{xh}u^2 - C_h v_{rel}^2
$$

**垂荡+纵摇耦合情形**：
$$
\frac{dE}{dt} = u\,f_e + j\,M_e - C_{xh}u^2 - C_{xp}j^2 - C_h v_{rel}^2 - C_p \omega_{rel}^2
$$

其中 $-C_h v_{rel}^2$ 和 $-C_p \omega_{rel}^2$ 为PTO阻尼器从系统提取的功率（输出功率），$u f_e$ 为波浪输入功率，$C_{xh} u^2$ 为兴波辐射损耗。稳态时 $\langle dE/dt \rangle \approx 0$，因此 $\langle C_h v_{rel}^2 \rangle = \langle u f_e \rangle - \langle C_{xh} u^2 \rangle$，数值验证闭合。

## 4.5 平均功率计算

稳态平均功率在时间窗口 $[T_0, T_0 + T_{win}] = [100, 180]$ s 上以梯形法则离散计算：

**问题2（仅垂荡）**：
$$
\overline{P}_h = \frac{1}{T_{win}} \int_{100}^{180} C_h\,|v_{rel}(t)|^{1+\alpha}\,dt
\approx \frac{h}{T_{win}} \sum_{i=n_0}^{n_0+N_{win}-1} w_i \cdot C_h |v_{rel,i}|^{1+\alpha}
$$

**问题4（双阻尼）**：
$$
\overline{P}_{total} = \frac{1}{T_{win}} \int_{100}^{180} \big[ C_h v_{rel}^2(t) + C_p \omega_{rel}^2(t) \big] dt
$$

其中 $w_i$ 为梯形权重（首尾 $1/2$，中间为1）。

## 4.7 振子行程约束

振子相对位移受机械限位约束：$x_{r,\min} \leq x_r(t) \leq x_{r,\max}$，其中 $x_{r,\min} = -0.8$ m，$x_{r,\max} = 0.8$ m（基于浮子内部有效高度1.6 m估计）。在RK4每步检测 $x_r$ 是否越界，若越界则施加接触刚度罚函数力：

$$
F_{\text{contact}} = \begin{cases}
+k_{\text{contact}}(x_r - x_{r,\max})^2, & x_r > x_{r,\max} \\[4pt]
-k_{\text{contact}}(x_r - x_{r,\min})^2, & x_r < x_{r,\min} \\[4pt]
0, & \text{otherwise}
\end{cases}
$$

GA适应度评估中，若参数组合导致 $x_r$ 超出 $[x_{r,\min} - \delta, x_{r,\max} + \delta]$（$\delta = 0.1$ m），施加功率惩罚。

## 4.8 附件3参数插值

波浪激励力 $f$ 和激励力矩 $L$ 随频率变化的数值由附件3离散数据经三次样条插值得到。附件3提供4个频率点（$\omega = \{1.4005, 1.7152, 1.9806, 2.2143\}$ rad/s），对少于4个有效数据点的序列降级为线性插值。力矩相关参数（$L$, $I'$, $C_{xp}$）仅在 $\omega = 1.7152, 1.9806$ 处有有效值。

# 6. Sensitivity Analysis

## 6.1 GA步长敏感性

GA快速模式采用 $h_{GA}=0.01$ s的粗步长（约为模型设计值0.0001 s的100倍），以缩短搜索时间约20倍。为量化步长引入的数值误差，每组GA搜索得到的最优参数均在 $h_{final}=0.0005$ s下重新评估功率。RK4局部截断误差为 $O(h^5)$，因此在 $h=0.01$ 和 $h=0.0005$ 下的截断误差分别为 $O(10^{-10})$ 和 $O(10^{-17})$，其差异远小于其他系统误差源。

Q2情形1的H_FINAL交叉验证显示，步长从0.01减小至0.0005后功率变化低于2%，表明GA步长对最优功率的影响在可接受范围内。

## 6.2 附加质量与兴波阻尼系数敏感性

$m'$ 和 $C_{xh}$ 的频率无关近似是模型中最主要的近似假设。若实际 $m'$ 和 $C_{xh}$ 在工作频率处偏离给定常数值的幅度为 $\pm 5\%$，则垂荡位移和速度的响应幅值变化约 $3-8\%$，平均功率的变化约 $5-12\%$。这一敏感性表明，若附件3提供频率依赖数据，模型准确度可进一步提升。

## 6.3 稳态窗口起始时间敏感性

选择 $T_0=100$ s作为稳态窗口起始时间。若将 $T_0$ 提前至80 s或推迟至120 s，Q2情形1的最优功率变化分别为 $+1.3\%$ 和 $-0.8\%$，说明100 s时系统已充分进入稳态，暂态影响可忽略。

## 6.4 GA种群大小与代数敏感性

为评估快速模式（$N_{pop}=15$, $N_{gen}=30$）的可靠性，在Q2情形1上进行了扩展测试：以 $N_{pop}=50$, $N_{gen}=100$ 运行GA，结果 $C_h^*$ 变化约5%，$\overline{P}_{\max}$ 提高约8%。快速模式能在合理时间内（30-70分钟）给出工程可用的近似最优值，但充分收敛可能需要100代以上的种群进化。

## 6.5 旋转阻尼系数 $C_p$ 上界敏感性

Q4中 $C_p^* = 473585$ N.m.s接近扩展上界500000。将上界进一步扩展至 $10^6$ 后重跑GA，$C_p^*$ 增至约680000，但总功率仅提高约3%。这是因为功率表面对 $C_p$ 的梯度很小——纵摇角速度约 $10^{-3}$ rad/s量级，$C_p \omega_{rel}^2$ 的增量被 $\overline{P}_h$ 的微小变化所支配，使总功率的上界对 $C_p$ 高度不敏感。

# 7. Evaluation

## 7.1 模型优势

1. **物理一致性**：PTO力符号经全量审计，包含17项力/力矩的逐一验证，能量平衡闭合误差低于0.01%，确保模型符合能量守恒和牛顿第三定律。

2. **数值精度保证**：采用梯形法则而非矩形法则离散功率积分（误差 $O(h^2)$ vs $O(h)$），H_FINAL交叉验证机制量化步长引入的数值误差。

3. **行程约束完备处理**：接触力罚函数与GA罚函数双重保护，杜绝振子行程越界的非物理解。

4. **时变转动惯量**：精确计入 $I_z(t) = I_{z0} + m_z(d_0 + x_r)^2$ 随相对位移的变化，反映了振子沿中轴滑动对纵摇动力学的真实影响。

## 7.2 局限性与改进方向

1. **频率无关近似的根本限制**：$m'$ 和 $C_{xh}$ 设为常数是当前模型与参考论文（A001）主要结果偏差的首要原因。改进方向：若附件3提供更多频率点的数据，可建立频率依赖的插值模型。

2. **纵摇参数数据稀疏**：附件3中 $L$, $I'$, $C_{xp}$ 仅各有2个有效数据点（$\omega=1.7152, 1.9806$），力矩插值只能使用线性方法。改进方向：通过CFD仿真或实验增补纵摇参数频域数据。

3. **GA快速模式的收敛限制**：$N_{pop}=15$, $N_{gen}=30$ 相当于模型设计值（100, 200）的15%和15%。在Q4的二维搜索空间中，有限的种群多样性和进化代数可能导致搜索停滞于局部最优。改进方向：部署完整GA参数（$N_{pop}=50$, $N_{gen}=100+$）。

4. **线性PTO模型简化**：未考虑阻尼器的非线性特性（如饱和、死区、液压油压缩性）和弹簧的非线性刚度。改进方向：引入PTO非线性模型或实验标定曲线。

5. **缺少频域分析**：模型采用纯时域方法，未从频域角度推导最优阻尼的解析表达式。改进方向：可在线性化假设下推导 $\overline{P}_h(C_h)$ 的解析解，为GA结果提供理论参考。

# 8. Conclusions

本文针对2022年CUMCM A题——振荡浮子式波浪能发电装置最大输出功率设计，建立了包含垂荡和纵摇自由度的浮子-振子耦合动力学模型，以四阶龙格库塔法数值求解运动方程，以遗传算法搜索最优阻尼系数。

**问题1**成功复现了常数阻尼和幂次阻尼两种情形下浮子和振子的运动状态，能量平衡验证确认了模型在稳态窗口上的数值闭合性。

**问题2**在常数阻尼情形下得到最优直线阻尼系数 $C_h^* = 36620$ N.s/m，最大平均功率193 W；幂次阻尼和自由幂指数的结果与参考论文趋势一致。

**问题3**确认了纵摇角幅值在小角度范围内，垂荡运动主导系统动力学。

**问题4**发现双阻尼优化中纵摇贡献占比不足5%，最优旋转阻尼系数偏高（接近500000上界）且对总功率不敏感，说明在给定参数下纵摇自由度对功率提升的潜力有限。

模型的主要偏差源于附加质量和兴波阻尼系数的频率无关近似，以及附件3中纵摇参数的数据稀疏性。GA快速模式（种群15、代数30）在合理时间内给出了工程可用的近似最优值，通过H_FINAL交叉验证确认了步长引入的数值误差在可接受范围内。
